How to Add a COTI V2 Token to MetaMask

Step 1: Go to COTI Token Explorer

Visit mainnet.cotiscan.io/tokens

Step 2: Search and Select Your Token

Type the name of the token you want to import and click on it to open the token page.

Step 3: Copy the Token Address

On the token's page, copy the token contract address to your clipboard.

Step 4: Open MetaMask

Launch your MetaMask extension or mobile app.

Step 5: Click the 3 Dots and Choose "Import Tokens"

In the Tokens tab, tap the 3 dots icon, then select Import Tokens.

Step 6: Select Network & Paste Address

Select the COTI network, then paste the token address into the input field.

Step 7: Click Next

Click the Next button to continue.

Step 8: Click Import

Click Import to finish adding the token.

Step 9: Done!

The token should now appear in your MetaMask wallet.
